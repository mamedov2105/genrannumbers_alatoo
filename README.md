# 🎲 Random List Generator

Простой Python-проект для генерации списка случайных чисел.

## 📌 Описание

Этот проект позволяет создавать список случайных чисел в заданном диапазоне.
Можно генерировать как обычные числа, так и уникальные.

## 🚀 Возможности

* Генерация списка случайных чисел
* Настройка диапазона (min / max)
* Указание количества чисел
* Поддержка уникальных значений

## 🛠️ Технологии

* Python 3
* Стандартная библиотека (`random`)

## ▶️ Запуск проекта

1. Клонировать репозиторий:

```bash
git clone https://github.com/mamedov2105/genrannumbers_alatoo
```

2. Перейти в папку проекта:

```bash
cd genrannumbers_alatoo
```

3. Запустить:

```bash
python main.py
```

## 📄 Пример использования

```python
from generator import generate_random_list

numbers = generate_random_list(1, 100, 10)
print(numbers)
```

## ⚙️ Параметры функции

| Параметр  | Описание                         |
| --------- | -------------------------------- |
| min_value | Минимальное значение             |
| max_value | Максимальное значение            |
| count     | Количество чисел                 |
| unique    | Уникальные значения (True/False) |

## 💡 Пример вывода

```

```
